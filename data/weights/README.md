# 模型权重目录

本目录存放 YOLO 三阶段级联检测管线的模型权重文件。

## 目录结构

```
data/weights/
├── README.md               ← 本文件（Model Card 汇总）
├── stage_one/              ← 阶段一：目标检测（全图→车辆 BBox）
│   ├── stage_one.pt
│   ├── stage_one.onnx
│   └── stage_one.engine     ← TensorRT FP16，部署时生成，不入 Git
├── stage_two/              ← 阶段二：装甲板颜色 + 编号分类
│   ├── stage_two.pt
│   ├── stage_two.onnx
│   └── stage_two.engine
└── stage_three/            ← 阶段三：灰色装甲板分类
    ├── stage_three.pt
    ├── stage_three.onnx
    └── stage_three.engine
```

## 文件格式说明

| 后缀      | 用途               | Git 管理方式          |
|-----------|--------------------|-----------------------|
| `.pt`     | PyTorch 原始权重    | Git LFS 追踪          |
| `.onnx`   | ONNX 中间格式       | `.gitignore` 排除     |
| `.engine` | TensorRT FP16 引擎  | `.gitignore` 排除     |

> `.engine` 文件与 GPU 型号、TensorRT 版本强绑定，不可跨设备复用。  
> 部署时通过 `scripts/export_models.py` 从 `.pt` 重新导出即可。
